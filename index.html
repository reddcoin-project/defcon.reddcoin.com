Preamble

Welcome to the DefCon29 Reddcoin CTF event.

This is the first CTF event that the Reddcoin Team has had the honour to participate in.
For the scope of this event we are targeting a couple of different technologies that make up the Reddcoin environment.
There are flags available in the following categories.

  • Reddcoin general knowledge.
  • Browser extensions.
  • Blockchain.

In order to participate, there are a few tools available to you which we have outlined below.

We also provide the following documentation to aid you in your journey.

Browser extension

The browser extension is configured to run on the electrum network (testnet is already enabled for the extension).

You can download the git repo or the packaged file from the releases (links provided below).

Using your browsers extension loader,

Chrome: Extensions can be loaded in unpacked mode by following the following steps:

  1. Visit chrome://extensions (via omnibox or menu -> Tools -> Extensions).
  2. Enable Developer mode by ticking the checkbox in the upper-right corner.
  3. Click on the "Load unpacked extension..." button.
  4. Select the directory containing your unpacked extension.

If you have a crx file, then it needs to be extracted first. CRX files are zip files with a different header. Any capable zip program should be able to open it.

Firefox: Extensions can be loaded temporarily (until reload or removed) by following the following steps:

  1. open Firefox
  2. enter "about:debugging" in the URL bar
  3. click "This Firefox"
  4. click "Load Temporary Add-on"
  5. open the extension's directory and select any file inside the extension,
    or select the packaged extension (.zip file).

Electrum Wallet

Check out the code from GitHub:

  • git clone git://github.com/reddcoin-project/electrum-redd.git
  • cd electrum-redd
  • git submodule update --init
  • python -m venv myenv
  • source myenv/bin/activate
  • Follow the readme.rst for installation of dependencies

Reddcoin Core Wallet

Download the appropriate version from website

  • Download and install to your system
  • Blockchain sync can take some incredible time, we have provided bootstraps for both mainnet and testnet
    • Download the bootstrap and extract to the required path
